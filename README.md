# Portfolio Website

Personal portfolio website showcasing cybersecurity and hydrography expertise.

## Tech Stack

- HTML5 / CSS3 / JavaScript
- No frameworks or dependencies
- Google Fonts (Outfit, Space Mono)

## Features

- Trilingual support (English, French, Spanish)
- Dark/Light theme toggle with system preference detection
- Responsive design
- Scroll animations
- SVG logo

## Structure
```
├── index.html          # English version
├── fr/index.html       # French version
├── es/index.html       # Spanish version
├── css/styles.css      # Global styles
├── js/main.js          # Theme toggle & animations
└── assets/             # Images, logo, CVs
```
